# Global Football News

This project is a simple responsive football news web page with a Google search box, PWA support, and local live preview.

## Run locally

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000

## Free hosting options

- GitHub Pages
- Netlify
- Vercel

## Deploy to GitHub Pages

1. Push this project to a GitHub repository.
2. Open the repository on GitHub and go to Settings > Pages.
3. Under Source, choose GitHub Actions.
4. The included workflow will build and publish the site automatically on every push to the main branch.

The site is already configured as a static page, so it can be hosted directly from the repository.
